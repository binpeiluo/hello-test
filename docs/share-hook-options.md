# Multica Number Merge - Share Hook Options

## 分享机制设计

### 分享时机

#### 1. 关卡完成时（最佳时机）

**触发条件：** 完成关卡目标
**分享内容：**
- 关卡号和星级评价
- 最终分数
- 剩余步数
- 连击记录

**文案示例：**
- "我在第 15 关拿到了 3 星！你能超过我吗？"
- "5000 分通关第 15 关，挑战一下？"
- "还剩 10 步就通关了，你能更少吗？"

**设计原则：**
- 自然的成就分享
- 包含挑战元素
- 不强迫分享

#### 2. 每日挑战完成时

**触发条件：** 完成每日挑战
**分享内容：**
- 今日挑战分数
- 全球排名百分比
- 连续挑战天数

**文案示例：**
- "今日挑战 8500 分，超过 95% 的玩家！"
- "连续 7 天完成每日挑战！"
- "今天的挑战你能得多少分？"

**设计原则：**
- 每日新鲜感
- 竞争元素
- 习惯养成

#### 3. 达成特殊成就时

**触发条件：** 达成特定成就
**分享内容：**
- 成就名称
- 达成条件
- 稀有度

**文案示例：**
- "达成「连击大师」成就！连续 5 次消除！"
- "解锁「数字猎人」成就！合成 1024！"
- "获得「完美通关」成就！所有关卡 3 星！"

**设计原则：**
- 成就感
- 稀有性
- 收集欲

#### 4. 有趣的残局局面

**触发条件：** 特殊棋盘状态
**分享内容：**
- 当前棋盘截图
- 当前分数
- 挑战问题

**文案示例：**
- "这个局面怎么解？来试试！"
- "差一步就合成 512 了！"
- "这个连击你能做到吗？"

**设计原则：**
- 趣味性
- 挑战性
- 互动性

### 分享渠道

#### 1. 微信好友

**场景：** 直接分享给好友
**优势：**
- 精准触达
- 私密性好
- 不打扰

**设计：**
- 选择特定好友
- 附带个性化消息
- 可选附带残局

#### 2. 微信群

**场景：** 分享到游戏群
**优势：**
- 覆盖面广
- 竞争氛围
- 社交互动

**设计：**
- 选择特定群
- 附带挑战信息
- 可选附带残局

#### 3. 朋友圈

**场景：** 公开分享成就
**优势：**
- 最大曝光
- 社交认同
- 长期展示

**设计：**
- 精美截图
- 成就标签
- 避免刷屏

### 分享文案库

#### 挑战类

1. "我在第 {level} 关拿到了 {score} 分，你能超过我吗？"
2. "今日挑战 {score} 分，超过 {percent}% 的玩家！"
3. "连续 {days} 天完成每日挑战！"
4. "达成 {achievement} 成就，你能吗？"
5. "{level} 关 {stars} 星通关，挑战一下？"

#### 趣味类

1. "这个局面怎么解？来试试！"
2. "差一步就合成 {value} 了！"
3. "这个连击你能做到吗？"
4. "看看这个有趣的数字组合！"
5. "你能用 {moves} 步通关吗？"

#### 成就类

1. "解锁「{achievement}」成就！"
2. "所有关卡 3 星通关！"
3. "累计合成 {count} 个数字！"
4. "最高合成 {value}！"
5. "总分达到 {score}！"

**风格：**
- 新鲜感
- 竞争性

#### 成就图

**元素：**
- 成就图标
- 成就名称
- 达成条件
- 稀有度标签

**风格：**
- 成就感
- 稀有性

### 分享反打扰设计

#### 避免的文案

1. "帮我复活！" - 强迫求助
2. "我需要体力！" - 强迫求助
3. "快来帮我！" - 强迫求助
4. "不分享就失败！" - 强迫分享

#### 推荐的文案

1. "我拿到了 5000 分，你能超过吗？" - 挑战
2. "这个局面怎么解？" - 趣味
3. "今日挑战你能得多少分？" - 竞争
4. "解锁新成就！" - 成就

### 分享数据追踪

#### 追踪指标

1. **分享率：** 触发分享时机的比例
2. **点击率：** 收到分享后点击的比例
3. **回流率：** 点击分享后进入游戏的比例
4. **留存率：** 通过分享进入的用户留存率

#### 优化方向

1. **文案优化：** 测试不同文案的点击率
2. **时机优化：** 测试不同分享时机的触发率
3. **图片优化：** 测试不同图片的吸引力
4. **渠道优化：** 测试不同渠道的传播效果

### 实现优先级

#### P0（MVP 必须）

1. 关卡完成分享
2. 微信好友分享
3. 基础分享文案

#### P1（重要功能）

1. 每日挑战分享
2. 微信群分享
3. 分享图片生成

#### P2（增强功能）

1. 成就分享
2. 朋友圈分享
3. 残局分享
4. 分享数据追踪

### 总结

分享机制的核心是**自然、有趣、不打扰**。通过成就、挑战、趣味三个维度，在合适的时机提供自然的分享入口，让用户愿意主动分享，而不是被强迫分享。

**关键成功因素：**
1. 分享时机自然
2. 分享内容有趣
3. 分享体验流畅
4. 不影响核心游戏体验
