# Multica Number Merge - Art Direction

## 视觉风格概述

### 设计理念

**核心理念：** 温暖、清晰、有反馈

**设计原则：**
1. **简洁：** 去除不必要的视觉元素
2. **清晰：** 数字和目标一目了然
3. **温暖：** 使用温暖的色彩和圆润的形状
4. **反馈：** 每个操作都有视觉反馈

### 目标用户

- 年龄：25-45 岁
- 偏好：简洁、舒适、不刺眼
- 场景：碎片时间、休闲娱乐

## 动画设计

### 方块移动动画

**时长：** 150ms
**缓动：** ease-out
**效果：** 平滑位移

### 方块合成动画

**时长：** 200ms
**缓动：** ease-in-out
**效果：**
1. 方块放大 1.2 倍（100ms）
2. 方块缩小回原大小（100ms）
3. 颜色变化

### 方块消除动画

**时长：** 300ms
**缓动：** ease-in
**效果：**
1. 方块闪烁（150ms）
2. 方块缩小并消失（150ms）

### 连击特效

**时长：** 500ms
**缓动：** ease-out
**效果：**
1. 特效粒子扩散
2. 分数飘字上升
3. 连击数字显示

### 胜利特效

**时长：** 1000ms
**缓动：** ease-out
**效果：**
1. 烟花粒子效果
2. 星星闪烁
3. 胜利文字弹出

### 失败特效

**时长：** 500ms
**缓动：** ease-in
**效果：**
1. 灰色遮罩淡入
2. 失败文字弹出
3. 按钮淡入

## 总结

视觉设计的核心是**温暖、清晰、有反馈**。通过合理的色彩、字体、形状、动画和音效设计，创造一个舒适、清晰、有反馈的游戏体验。

**关键成功因素：**
1. 色彩温暖不刺眼
2. 数字清晰易读
3. 动画流畅有反馈
4. 音效愉悦不干扰
5. 布局合理易操作
