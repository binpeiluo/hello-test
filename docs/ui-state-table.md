## 状态转换规则

### 1. 应用状态转换

```
启动 → 首页 → 关卡选择 → 游戏中 → 暂停 → 游戏中
                ↓           ↓
              设置        胜利/失败
```

### 2. 游戏状态转换

```
空闲 → 移动中 → 合成中 → 消除中 → 空闲
  ↓                              ↓
游戏结束                      游戏结束
```

### 3. 方块状态转换

```
无方块 → 有方块 → 合成中 → 有方块（新数值）
  ↓                ↓
消除中 → 无方块
```

### 4. 目标进度转换

```
未开始 → 进行中 → 即将完成 → 已完成
```

## 状态动画

### 1. 方块移动动画

- 时长：150ms
- 缓动：ease-out
- 效果：平滑位移

### 2. 方块合成动画

- 时长：200ms
- 缓动：ease-in-out
- 效果：放大 + 颜色变化

### 3. 方块消除动画

- 时长：300ms
- 缓动：ease-in
- 效果：闪烁 + 消失

### 4. 连击特效

- 时长：500ms
- 缓动：ease-out
- 效果：特效 + 分数飘字

### 5. 胜利特效

- 时长：1000ms
- 缓动：ease-out
- 效果：烟花特效

### 6. 失败特效

- 时长：500ms
- 缓动：ease-in
- 效果：灰色遮罩

## 状态管理

### 1. 状态存储

- 应用状态：内存
- 游戏状态：内存
- 进度状态：本地存储
- 设置状态：本地存储

### 2. 状态同步

- 游戏状态：实时同步
- 进度状态：游戏结束时同步
- 设置状态：修改时同步

### 3. 状态恢复

- 应用状态：启动时恢复
- 游戏状态：从存储恢复
- 进度状态：从存储恢复
- 设置状态：从存储恢复
